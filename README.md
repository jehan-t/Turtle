# Turtle Graphics Art Generator

This project is a Python program that uses the `turtle` module to create
randomized polygon art. The user can select from **1 to 9** different
art modes, and each mode generates shapes with random sizes, colors,
orientations, and locations. The project demonstrates objectâ€‘oriented
programming, randomization, and graphical output using Python.

## How the Program Works

The program defines a `Polygons` class that handles: - Randomizing
polygon attributes\
- Drawing single polygons\
- Drawing nested polygons\
- Creating multiple randomized shapes\
Each mode uses different settings to produce a unique style of artwork.

## Art Mode Overview

-   **1** -- Triangle outlines\
-   **2** -- Square outlines\
-   **3** -- Pentagon outlines\
-   **4** -- Random-side outline polygons\
-   **5** -- Nested triangles\
-   **6** -- Nested squares\
-   **7** -- Nested pentagons\
-   **8** -- Random nested polygons\
-   **9** -- Mixed outline + nested shapes
